### 修改placeholder样式

```css
input::-webkit-input-placeholder{
    color: #A9A9A9;
    font-size: 14px;
    letter-spacing: 1px;
}
input::-moz-placeholder{
    color: #A9A9A9;
    font-size: 14px;
    letter-spacing: 1px;
}
input::-ms-input-placeholder{
    color: #A9A9A9;
    font-size: 14px;
    letter-spacing: 1px;
}
```
